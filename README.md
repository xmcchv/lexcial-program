# lexcial-program
编译原理词法分析，语法分析，语义分析与中间代码生成

## 现在它能做什么？

1. 词法分析  路径：morphology/lexical.java
2. 语法分析  路径：morphology/Grammer.java
3. 语义分析以及生成四元式 路径：morphology/semantic.java
4. 带界面的预测分析法（有bug） 路径：predictive/prelexical.java
